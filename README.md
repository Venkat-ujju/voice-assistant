# Voice Assistant

This is a basic Python voice assistant that listens to your voice commands, opens websites, sends OTP emails, and responds with speech. Itâ€™s a fun project to learn how to combine voice recognition, automation, and email features using Python.

## Features

- Recognizes voice commands  
- Opens websites like YouTube, LinkedIn, GitHub  
- Sends OTPs securely via email  
- Responds with text-to-speech

## Technologies Used

- Python  
- speech_recognition  
- pyttsx3  
- smtplib and ssl  
- webbrowser

## How to Run

1. Clone this repo:
   git clone https://github.com/your-username/voice-assistant.git
   cd voice-assistant
   
2.Install dependencies:

pip install speechrecognition pyttsx3
Run the assistant:

3.python voice_assistant.py

Note: Update the email credentials in the script before running to enable OTP sending.
